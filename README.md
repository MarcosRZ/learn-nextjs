# learn-nextjs
My [Zeit's NextJS tutorial](https://nextjs.org/learn/) implementation, with **babel / ES6** steroids.

### Install
```sh
$ npm i
```

### Generate production build
```sh
$ gulp build
```

### Run (Development mode)
```sh
$ npm run dev
```

### Run (Production mode)
```sh
$ gulp build
$ cd /dist
$ npm i
$ npm run prod
```
See _package.json_ scripts section for more information
